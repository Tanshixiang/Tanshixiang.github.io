# 欢迎来到小谭的空间

欢迎访问我的个人网页，这是一个简单的HTML页面，展示了如何使用背景图片和标题。

## 特点

- 响应式设计
- 简洁的布局
- 易于定制

## 使用方法

要使用这个网页，你需要做以下几步：

1. 下载或克隆这个项目到你的本地环境。
2. 将 `background.jpg` 替换成你想要用作背景的图片文件。
3. 根据需要修改 `index.html` 中的标题或其他内容。

### 技术栈

- HTML5
- CSS3

### 浏览器兼容性

这个页面应该在所有现代浏览器上都能正常工作，包括但不限于：

- Google Chrome
- Mozilla Firefox
- Safari
- Microsoft Edge
